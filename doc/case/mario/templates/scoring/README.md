# Mario 案例评分模板索引

这些模板用于 `score_required=true` 的命令，要求 reviewer 按模板给分并输出：
- `criteria_version`
- `score`（0-100）
- `score_breakdown`（分项得分与扣分原因）

模板列表：
- `doc/case/mario/templates/scoring/mario_planning_dag_review_rubric.v1.md`：规划阶段（DAG评审）评分量表
- `doc/case/mario/templates/scoring/mario_requirements_rubric.v1.md`：需求/验收标准评分量表
- `doc/case/mario/templates/scoring/mario_mvp_playable_rubric.v1.md`：执行阶段（可玩MVP验收）评分量表
- `doc/case/mario/templates/scoring/mario_release_readiness_rubric.v1.md`：发布准备度评分量表
